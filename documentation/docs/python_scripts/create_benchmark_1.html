create_benchmark¶

The create_benchmark.py script enables the automatic generation of datasets, based on the Cyber-Physical Process Plant setups saved in the benchmark_config.json file.

The script sequentially calls the following functionalities:
  • create_ds_dir(setup: str)

    creating a dataset directory, which is named after the dataset name defined in the benchmark_config.json and in which the results will be saved.

  • create_call_file(setup: str)

    creating a customized call.mos file for running the custom simulation and saving the results in a custom directory.

  • create_process_plant(config_ds: dict)

    creating a customized process_plant.mo file, based on the Cyber-Physical Process Plant defined in the benchmark_config.json.

  • create_supermodel(faulty_module: str, fault: str)

    creating a customized superModel of the faulty module, to induce time-dependent faults.

  • run_simulation(setup: str, fault: str)

    run individual simulations for all faults indicated in the benchmark_config.json.

  • clean_ds(setup: str, fault: str)

    clean the dataset from unnecessary values and only keep values of interest.

  • save_connection_model(config_dict: dict, setup: str)

    saving the Cyber-Physical Process Plant configuration as *.json in the custom dataset directory.

  • create_dataset(config_ds: dict, setup: str)

    running all functionality calls in a function.

     1def create_supermodel(faulty_module: str, fault: str):
 2    """
 3    Within the supermodels fault induction can be turned off and on.
 4    Hence, for each fault setup a separate supermodel must be created.
 5    """
 6    str_superModel = "../simulation_models/" + faulty_module[:-9] + "_superModel.mo"
 7    ok_str1 = "valve_leaking_simulator = if time >= 4000 then 0.0001 else 0; // leaking"
 8    nok_str1 = "valve_leaking_simulator = if time >= 2000 then 0.0001 else 0; // leaking"
 9    ok_str2 = "valve_clogging_simulator = if time >= 4000 then 0.01 else 1; // clogging"
10    nok_str2 = "valve_clogging_simulator = if time >= 2000 then 0.01 else 1; // clogging"
11
12    # read mode
13    file = open(str_superModel, "r")
14    replacement = ""
15    for line in file:
16        if fault == "l":
17            if ok_str1 in line:
18                changes = line.replace(ok_str1, nok_str1)
19            elif nok_str2 in line:
20                changes = line.replace(nok_str2, ok_str2)
21            else:
22                changes = line
23        elif fault == "c":
24            if nok_str1 in line:
25                changes = line.replace(nok_str1, ok_str1)
26            elif ok_str2 in line:
27                changes = line.replace(ok_str2, nok_str2)
28            else:
29                changes = line
30        elif fault == "lc":
31            if ok_str1 in line:
32                changes = line.replace(ok_str1, nok_str1)
33            elif ok_str2 in line:
34                changes = line.replace(ok_str2, nok_str2)
35            else:
36                changes = line
37        replacement = replacement + changes
38    file.close()
39    # write mode
40    fout = open(str_superModel, "w")
41    fout.write(replacement)
42    fout.close()

     1def create_dataset(config_ds: dict, setup: str):
 2    """
 3
 4    This funcion creates a single .csv dataset by
 5        - building a modelica model and supermodel of the desired process plant setup (including the adapted parametrization)
 6        - running the simulation (regarding to the desired fault setup)
 7        - cleaning the dataset from unnecessary values
 8        - and saving it in a separate directory
 9
10    in:
11        - a dict of the current ds config
12
13    returns:
14        - a csv in a directory named after the process plant build
15    """
16
17    create_ds_dir(setup)                            # creating a directory for the dataset setup
18    create_call_file(setup)                         # creating an adapted call.mos file to run Modelica simulations
19    create_process_plant(config_ds)                 # creating a process plant setup following the config file
20
21    faulty_module = config_ds["faulty_module"][0]   # choose module in which faults will be induced
22    for fault in config_ds["faults"]:
23        create_supermodel(faulty_module, fault)     # creating a supermodel inducing the fault "fault"
24        run_simulation(setup, fault)                # running the om simulation
25        clean_ds(setup, fault)                      # cleaning resulting ds from unnecessary values
26        print("simulation run " + str(setup) + str(fault) + " finished ... \n")
27    save_connection_model(config_ds, setup)         # saving the connection model and simulation setup as file in ds-directory
